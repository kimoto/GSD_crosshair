概要
・DirectX使用のソフトにcrosshairを描画します

仕組み
・GSDライブラリを利用してDirectXのAPIを中継し、任意の画像データを上書きしています

デフォルト設定での使い方
1. GSD_crosshair.exeを起動します
2. 開始ボタンをクリックします
	crosshairがアクティブなDirectXソフトにて描画されます
3. 停止ボタンをクリックするとcrosshairが消えます

crosshairの変更方法
1. crosshair.pngを画像編集ソフトで書き換えます(MS純正のペイントだと透過情報が保存できないのでよくないです)
	GIMPとかPaint.net推奨
2. GSD_crosshair.exeを起動します
3. 開始ボタンをクリックします
